Microsoft 2FA Verification Guide

⚠️ IMPORTANT: Do NOT message for a verification code until you are already on the Microsoft verification screen inside a Minecraft launcher.

Verification codes are intended ONLY for launcher login. If you request a code too early, allow it to expire, or use it outside of a launcher login, you will need to wait 24 hours before another code can be provided.
⏳ Codes expire in approximately 30 seconds. Only request one when you are ready to enter it immediately.

➡️ How to Use Your Account

  1. Open the official Minecraft Launcher (or another Minecraft launcher).
  2. Click Sign in with Microsoft.
  3. Enter the email address provided.
  4. Click Other ways to sign in → Use my password.
  5. If Microsoft asks for additional verification, select Authenticator App.
  6. Message the seller (Ibaddie) and request your verification code.
  7. Copy the code as soon as it is sent and enter it immediately into Microsoft.
  8. Login complete — enjoy!

⚠️ Important Rules

  • Only request a code when you're at the Microsoft verification screen.
  • Do NOT request a code "just to test."
  • Do NOT let the code expire.
  • Follow the login guide exactly.
If a code is requested too early or allowed to expire, another code cannot be provided for 24 hours.
