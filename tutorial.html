Login Guide

💡 About Your Account: This is a Full Access account with 2FA system — not NFA or minimal full access. You log in with just your email and password. Verification codes are only needed if Microsoft asks for additional verification.
⚠️ IMPORTANT: Do NOT request a verification code until you are already on the Microsoft verification screen inside a Minecraft launcher.

If you request a code too early, allow it to expire, or use it outside of a launcher login, you will need to wait before another code can be provided.
⏳ Codes expire in approximately 30 seconds. Only request one when you are ready to enter it immediately. Ibaddie must review your screenshot before sending the code.

➡️ How to Login

  1. Open the official Minecraft Launcher (or another Minecraft launcher).
  2. Click Sign in with Microsoft.
  3. Enter the email address and password provided.
  4. If Microsoft logs you in without asking for a code — you're done! Enjoy the game.
  5. If Microsoft asks for additional verification, select Authenticator App or Enter a code.
  6. Take a screenshot of the verification screen showing the code entry field.
  7. Open your Code Request Link (provided in your delivery message).
  8. Click Request Code and upload the screenshot you just took.
  9. Wait for Ibaddie to approve your request (usually under 1 minute when online). The code will appear on your screen.
  10. Enter the code immediately into the launcher — it expires in 30 seconds!

⚠️ Important Rules

  • Only request a code when you're at the Microsoft verification screen in a launcher.
  • Do NOT request a code "just to test."
  • Do NOT let the code expire.
  • Follow the login guide exactly.
