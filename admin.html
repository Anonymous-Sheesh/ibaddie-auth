🔧 Admin Tools
To test customer links without affecting their quota/cooldown:
  1. Open this discovery page on each of your devices (laptop, phone)
  2. Copy the HWID hash it shows
  3. Add it to the ADMIN_HWIDS env var in your Cloudflare Worker (comma-separated)
  4. Then access any customer link with &admin=1 appended — your access won't affect the buyer
You can also click View Code on any token below to see its current code without leaving this dashboard.
